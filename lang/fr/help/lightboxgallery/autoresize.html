Redimensionner des images

Vous pouvez controler de quelle manière les images de la galerie seront redimensionnées. Les méthodes suivantes sont accessible quand vous configurez la galerie :

  • Écran : les images qui sont plus grandes que l'écran de l'utilisateur seront réduites pour s'afficher totalement sur l'écran.
  • Déposer : les images seront redimensionnées aux dimensions spécifiées quand elles sont déposées par l'option "Ajouter une image".

Vous avez aussi accès à un plugin de redimensionnement d'image dans le mode édition de l'image, où vous pouvez manuellement redimensionner les images.
